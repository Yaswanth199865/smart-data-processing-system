# Smart Data Processing System

## Description
A Python Flask-based API that processes input data, validates it, and stores it in a SQLite database.

## Features
- REST API using Flask
- Data validation
- SQLite database integration
- Logging system

## Tech Stack
- Python
- Flask
- SQLite
